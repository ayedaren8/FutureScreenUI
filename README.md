# FutureScreen UI 前端数据可视化组件库
 
> 叶国庆(maxwellye)的本科毕业设计🚀
* 独自编纂，但未测试，仅供个人学习使用! 
## 安装 FutureScreen 
``` shell
npm install --save futurescreen
```

## 使用

### 全局引入

``` js 
import FSUI from 'future-screen' ;
import 'future-screen/style/index.css';
Vue.use(FSUI)
```

### 按需引入
``` js 
import {button} from 'future-screen' ;
import 'future-screen/style/index.css';
Vue.use(button)
```
